# HTML/CSS MCQ Test Application

A comprehensive 100-question multiple choice test application for HTML and CSS knowledge assessment.

## Features

- **100 Questions**: Covering both HTML and CSS fundamentals
- **Interactive Interface**: Clean, modern UI with smooth animations
- **Progress Tracking**: Visual progress bar and question tracking
- **Navigation**: Easy navigation between questions
- **Results Analysis**: Detailed score breakdown and performance analysis
- **Review Mode**: See correct answers with explanations
- **Responsive Design**: Works on desktop and mobile devices
- **Keyboard Support**: Navigate using arrow keys
